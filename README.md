# Music-Recorder
CSCC01 Final project

## Description
Records up to 3, 8 Note songs and can be played back
  - 12 different notes
  - 3 Songs
  - Play / Record
  - Display Songs on Monitor

## Setup
GPIO [3] \= Power
GPIO [5] \= Signal

SW [0:11] \= Notes
SW [12:13] \= Choose songs
SW [16] \= {Off \= Record, On \= Play} 

KEY [0] \= Start
